#### Find mounted drives

```sh
lsblk
```

#### Get UUID

```sh
sudo blkid /dev/sda1
```

#### Create mount location

```sh
sudo mkdir -p /mnt/flex
```

#### Change ownership

```sh
sudo chown -R $USER:$USER /mnt/flex && sudo chmod -R 755 /mnt/flex
```

#### Modify fstab

```sh
sudo nano /etc/fstab
```

#### Add line below other entries

```sh
UUID=22d23d78-272d-4d51-9168-3ac30bfb6353  /mnt/flex       ext4            auto,nofail,noatime,rw,users  0    0
```

#### Unmount drive

```sh
sudo umount /dev/sda1
```

#### Reboot

```sh
sudo reboot
```
