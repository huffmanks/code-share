## Create/change directory for mirror to Gitea

```sh
mkdir -p {{{DOCKER_PATH_VAR}}}/mirror-to-gitea && cd {{{DOCKER_PATH_VAR}}}/mirror-to-gitea
```

## Create docker-compose.yml

```sh
# *!* Copy content from next command before executing *!*
```
