<Steps>

1.  ## Prebuild binaries

        ```sh
        sudo apt install libgtk-4-bin libheif1 libraw-bin ffmpeg -y
        ```

2.  ## Add repo to apt source

        ```sh
        sudo add-apt-repository ppa:xtradeb/apps && sudo apt update
        ```

3.  ## Install

        ```sh
        sudo apt install czkawka
        ```

4.  ## Run GUI

        ```sh
        czkawka_gui
        ```

</Steps>
