<Steps>

1.  ## Install

        <Tabs syncKey="qtVersion">
            <TabItem label="Qt5">
            <Code lang="sh" wrap preserveIndent code="sudo apt update && sudo apt install qt5ct -y" />
            </TabItem>
            <TabItem label="Qt6">
            <Code lang="sh" wrap preserveIndent code="sudo apt update && sudo apt install qt6ct -y" />
            </TabItem>
        </Tabs>

        <br />

2.  ## Update environment

        <Code lang="sh" wrap preserveIndent code="sudo nano /etc/environment" />

        Add the following line to the file, then save and log out to apply changes:

        <Tabs syncKey="qtVersion">
            <TabItem label="Qt5">
            <Code lang="sh" wrap preserveIndent code="QT_QPA_PLATFORMTHEME=qt5ct" />
            </TabItem>
            <TabItem label="Qt6">
            <Code lang="sh" wrap preserveIndent code="QT_QPA_PLATFORMTHEME=qt6ct" />
            </TabItem>
        </Tabs>

        <br />

3.  ## Create custom palettes

        Create the configuration directory and paste your color definitions into a new `.conf` file.

        ### MacOSDark

        <Tabs syncKey="qtVersion">
            <TabItem label="Qt5">
            <Code lang="sh" wrap preserveIndent code="mkdir -p ~/.config/qt5ct/colors && nano ~/.config/qt5ct/colors/MacOSDark.conf" />
            </TabItem>
            <TabItem label="Qt6">
            <Code lang="sh" wrap preserveIndent code="mkdir -p ~/.config/qt6ct/colors && nano ~/.config/qt6ct/colors/MacOSDark.conf" />
            </TabItem>
        </Tabs>

        <Code lang="ini" wrap preserveIndent code={macOSDarkConf} />

        ### OneDarkPro

        <Tabs syncKey="qtVersion">
            <TabItem label="Qt5">
            <Code lang="sh" wrap preserveIndent code="nano ~/.config/qt5ct/colors/OneDarkPro.conf" />
            </TabItem>
            <TabItem label="Qt6">
            <Code lang="sh" wrap preserveIndent code="nano ~/.config/qt6ct/colors/OneDarkPro.conf" />
            </TabItem>
        </Tabs>

        <Code lang="ini" wrap preserveIndent code={oneDarkProConf} />

        <br />

4.  ## Apply settings

        Launch the graphical interface to select your new palette and set the “Fusion” style.

        <Tabs syncKey="qtVersion">
            <TabItem label="Qt5">
            <Code lang="sh" wrap preserveIndent code="qt5ct" />
            </TabItem>
            <TabItem label="Qt6">
            <Code lang="sh" wrap preserveIndent code="qt6ct" />
            </TabItem>
        </Tabs>

        **Note:** Within the tool, navigate to the **Appearance** tab.
            - Set the **Style** to `Fusion`.
            - Select **Custom** and choose your `Palette`.

</Steps>
