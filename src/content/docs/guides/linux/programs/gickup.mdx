<Steps>

1.  ## Download tarball

        ```sh
        wget https://github.com/stefanprodan/gickup/releases/download/v0.10.39/gickup_0.10.39_linux_amd64.tar.gz
        ```

2.  ## Verify checksum

        ```sh
        echo "c3d3e647189c4379d8d70b9223bd4fe5eb293b289ece5931ff2e10064b063a67  gickup_0.10.39_linux_amd64.tar.gz" | sha256sum -c -
        ```

3.  ## Extract and move

        ```sh
        tar -xzf gickup_0.10.39_linux_amd64.tar.gz && sudo mv gickup /usr/local/bin/gickup
        ```

4.  ## Make executeable

        ```sh
        sudo chmod +x /usr/local/bin/gickup
        ```

5.  ## Create backup/config directory

        ```sh
        mkdir -p ~/gickup-backups && mkdir -p ~/.config/gickup
        ```

6.  ## Create config

        ```sh
        nano ~/.config/gickup/config.yaml
        ```

        - Replace PATH_TO_LOCAL_BACKUP.
        - Replace YOUR_GITHUB_TOKEN.
        - Token requires these permissions:
                - `repo` - Full control of private repositories (required to access and clone private repos).
                - `gist` - Read user gists.
                - `read:org` - (Optional) Read-only access to organization membership and repos (needed if backing up org repos).



        <Code lang="yml" title="config.yaml" wrap preserveIndent code={yaml} />

7.  ## Run

        ```sh
        gickup ~/.config/gickup/config.yaml
        ```

</Steps>
