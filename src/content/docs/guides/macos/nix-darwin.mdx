## Getting started

<p>{frontmatter.description}</p>

1. Install Xcode Command Line Tools.

```sh
xcode-select --install
```

2. Run setup script.

```sh
bash -c "$(curl -fsSL {{{NIX_DARWIN_SETUP_SCRIPT_VAR}}})"
```

3. Reboot and enjoy your setup!

## Conclusion

By following this guide, you have configured macOS using nix-darwin to manage Finder and Dock settings, installed essential tools like Homebrew and nvm, and set up a reproducible development environment. Your system is now optimized for productivity and ease of use.
